# android-codelab-activity-lifecycle

Introduction
------------
- Android CodeLabで作ったアプリです。
- アプリ名称: Dessert Clicker
- Bundleにデザートの購入数、合計価格のデータを保存し、onDestroyしてもonCreateで前回のデータを再利用できます。

### アクティビティのライフサイクルのステージ ###
